How can you open the bookmarks panel? 
y Click the yellow bookmark button in the navbar
Press Alt+q (Windows/Linux) or option+q (Mac)
Search for 'bookmarks' in the project dropdown on the left

Which of the following can you do using the bookmarks panel:
y Copy a link to a filtered map or table for sharing
y Save a link to a filtered map or table to the bookmarks panel
y Apply preset filters to a map or table
Save a PNG of a map or table directly to disk

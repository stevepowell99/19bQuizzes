When you edit the quote for a link, what does the bracket notation `Actual quote [this bit is ignored] continues...` do?
y It lets you keep an exact quotation while leaving out an irrelevant passage in the middle
It adds a hint that only you can see
It marks the link as hypothetical
It deletes the bracketed text from the source
hint The text in square brackets is ignored, so the quote reads as a clean excerpt.

You type two influence factors and two consequence factors into a single link editor and save. How many links are created?
One
Two
y Four
hint Causal Map creates all the combinations: 2 causes x 2 effects = 4 links.

What does 'plain coding' create?
y A self-loop (cause = effect) tagged `#plain_coding`, marking that a theme is present rather than a causal claim
A link with no quote attached
A factor that cannot be edited
hint Plain coding marks 'theme present', and these self-loops can be hidden with the Exclude self-loops filter.

Link tags can carry caveats about how strong a claim is. Which prefix marks a 'weak' tag such as `?hypothetical` or `?future`?
A hash `#`
y A question mark `?`
An exclamation mark `!`
A tilde `~`

When you are typing a brand new factor label into a dropdown, how do you finish it so it is added?
y Type a comma or press tab
Press the spacebar twice
You cannot create new factors while coding
hint Finishing with a comma or tab commits the new factor.

What does Chain mode do while you are coding?
y It keeps the link editor open after you save, and uses your previous consequence factor as the next influence factor
It automatically codes the whole document for you
It locks the quote so it cannot be changed
It deletes the previous link

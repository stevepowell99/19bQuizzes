In a table, how do you sort by more than one column?
y Click one column header, then Shift+click the others
You can only ever sort by one column at a time; to sort by two you must export to Excel first
Double-click the table title
hint Shift+click adds extra sort columns; the most recent Shift+click is the primary sort.

In a prompt box, what happens to a line that begins with `//`?
y It is saved in your prompt history but not sent to the AI, so you can leave yourself notes
It is sent to the AI as a high-priority instruction that overrides the standard system prompt and any codebook rules you have set
It deletes the prompt
It is shown to other users
hint `//` lines are comments, for example '//Sarah's version with tweaks'.

How can you reorder the columns in a table?
y Drag the column headers into the order you want
Retype the data
You cannot reorder columns; column order is fixed and reflects the order fields were added to the project
hint Column headers can be dragged to rearrange them.

Prompts you write for AI coding, AI answers and similar features are:
y Auto-saved so you can browse and reuse them with the < and > buttons
Deleted as soon as you close the panel
Only kept if you manually save them each time by clicking the save icon next to the prompt field
Only visible to other users with edit access
hint Prompts auto-save and you can step back and forward through your history.

In a dropdown, how do you add an item that is not already in the list?
y Type part of the word and choose 'Create new...'
You cannot add new items to any dropdown
Right-click the dropdown and choose Insert
Press Ctrl+N to open a new item dialogue
hint Typing and selecting 'Create new...' adds an unlisted item.

How many rows per page can you choose to show in a table?
y 10, 25, 50 or 100
Only 10
Only all rows at once
hint Pagination offers 10, 25, 50 and 100 rows per page.

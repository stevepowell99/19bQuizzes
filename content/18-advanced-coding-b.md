How do you turn on AI coding in the app?
y Switch it on or off with a button in your account pane
Use the 'AI coding' switch just below the Sources bar
Email Causal Map support to enable it
Install a browser plugin

Causal Map's AI coding is described as 'human first, AI next'. What does this mean?
y The AI assists your coding rather than replacing your judgement
The AI codes everything first and you cannot change it
You must finish all coding by hand before the AI will run
The AI only works on documents written by humans, not transcripts

AI coding uses monthly credits. Which of these is true for a standard user?
y You get free credits each month and unused credits do not roll over
Credits are unlimited for all users
Credits never expire and accumulate month by month, so power users can build up a large reserve over a year
hint Standard users get a small monthly allocation (around 10 credits) that does not carry over.

Why should you use the source limit (for example 1, 5 or 20%) before running AI coding on everything?
y To test quality on a small sample before scaling up
Because the app cannot process more than 20% at a time
To save the remaining sources to a separate project
hint Work incrementally: check a sample, then scale up.

In the one-click pipeline, what runs when you have just a single source in scope?
y Only Auto-code runs
The full Auto-code, Revise codebook and Recode sequence runs
Nothing runs for a single source
hint Revise codebook and Recode only make sense across several sources.

What is the 'Background context' used for in AI coding?
y Shared project context that the AI coding prompts can draw on
The colour scheme of the map
A backup copy of your links, automatically compressed and stored in the cloud
A list of users who can see the file

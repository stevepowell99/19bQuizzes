In the statements panel, what does the Globe icon icon:globe do?
y It creates a link, quoting the whole of the statement
It repeats the previously quoted text for the influence factor field
It chains a new link after the old one using the same quote and consequence factor
It marks the claim as globally valid across all sources

In the coding panel, what does the Repeat icon icon:redo do?
y It quotes the same text again which you previously quoted
It quotes the whole statement and creates a fresh link from scratch
It chains a link onto the previous one and copies across both factor labels
It marks the claim as globally valid across all sources in the project

In the coding panel, the Chain icon icon:link; helps to chain a new link onto the end of the last one. How?
y It re-quotes the previous text and inserts your most recent consequence factor(s) into the influence dropdown
It inserts your most recent consequence factor(s) into the influence factor dropdown and saves the link automatically without further input
It re-quotes the previous text and guesses the next consequence factor based on what other coders have entered

What should you consider when creating a factor label?
Include as much detail as possible in the label
y Keep your factor labels brief
y Make sure you have the appropriate context

How can you get an overview of all the links coded for the current source?
Click the black "book" icon icon:book;
Click the green "book" icon icon:book;
Click on the Statements tab and filter by source_id
y Click on the Links tab and use filter by source_id

How would you code 'I got more money from sales of milk and also from selling my chickens'
y Put 'selling milk' and 'selling chickens' in the first box and 'got more money' in the second
Code this twice, once per cause factor, pasting the exact source quote separately into each link for full traceability

If you have a factor like `~Health behaviour; smoking` already in a factor dropdown when coding, how can you "flip" it to its opposite (`Health behaviour; ~smoking`)?
You can't; instead you must clear the field and retype the label with the tilde repositioned
y Click the icon with two opposing arrows icon:exchange above the factor dropdown
Click the globe icon icon:globe to reload the factor list from the server

If you want to construct a factor like `Health behaviour; eating fruit` in a factor dropdown when coding, and you have already coded the higher-level factor `Health behaviour`, how can you do this quickly?
You can't do this
You must type the full label: `Health behaviour; eating fruit`
y Select `Health behaviour` then type `; eating fruit` directly after it
Click the icon with two opposing arrows icon:exchange above the dropdown

To make all four links from both A and B to both C and D, A --> C, A --> D, B --> C, and B --> D:
Code each link separately
y Put A and B in the first box and C and D in the second
Use the special symbol `%%`
Click the icon with two opposing arrows icon:exchange above the dropdown

If you have already used `Health behaviour; eating fruit` while coding, but you have never used `Health behaviour` alone:
y Both `Health behaviour` and `Health behaviour; eating fruit` will appear in the factor dropdowns
Only `Health behaviour; eating fruit` will appear in the dropdowns, because `Health behaviour` was never entered as a standalone factor label
Only `~Health behaviour; eating fruit` will appear in the dropdowns, because the app strips the sub-label and inverts the parent automatically

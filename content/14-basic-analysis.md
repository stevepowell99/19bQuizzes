Which of these is true about coding: 
You must complete all your coding before using filters to analyse your map, because partial maps produce misleading results and the app prevents filtering until the final source is coded
y As you continue coding, you will sometimes use the filters to reflect on your work so far

**Map formatting** is used for the following:
To do calculations e.g. the robustness metric, which scores each factor's causal centrality across sources
y To apply overall formatting e.g. give all the factors a grey border
y To format links and factors to reflect the data e.g. scale links by citation count

Which of these is true about the Manage Links tab: 
You need to understand it to use Causal Map, because it is the only place where you can correct mistakes made during coding and reassign links to different sources
y Even beginners can have a look at the Manage Links tab to see what is going on, if they are interested
